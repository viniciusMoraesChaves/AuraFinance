# 📌 Sobre o Projeto

O **AuraFinance** é uma aplicação web focada em educação financeira e gestão de finanças pessoais.

A plataforma permite que usuários:

- acompanhem receitas e despesas
- visualizem sua situação financeira
- estabeleçam metas financeiras
- simulem investimentos
- analisem dados através de dashboards

O objetivo é facilitar o controle financeiro e promover decisões econômicas mais conscientes.

---

# ⚠️ Problema

Grande parte dos jovens adultos tomam decisões financeiras sem compreender seus impactos a médio ou longo prazo.

Isso resulta em:

- endividamento excessivo
- uso inadequado de crédito
- ausência de reserva financeira
- dificuldade de planejamento financeiro

O AuraFinance busca resolver esse problema oferecendo uma ferramenta simples, visual e educativa de gestão financeira.

---

# 👥 Público-Alvo

O público-alvo do aplicativo é formado por jovens adultos entre 18 e 35 anos, incluindo universitários, recém-formados e profissionais em início de carreira.

Esse grupo está em processo de conquista de independência financeira, geralmente com renda entre 1 e 5 salários mínimos, muitas vezes variável. Seus principais gastos concentram-se em moradia, alimentação, transporte, educação e lazer.

Apesar da alta familiaridade com tecnologia, muitos ainda possuem dificuldade em planejamento financeiro, controle de gastos e formação de reserva financeira.

O protótipo apresenta uma interface moderna, limpa e orientada a dados.

---

# 🏗️ Arquitetura do Sistema

A arquitetura inicial do MVP é baseada em três camadas principais:

Frontend (React)
↓
Backend API (Node.js)
↓
Banco de Dados (MySQL)

## Front-end
Responsável pela interface do usuário e experiência visual.

## Back-end API
Responsável por:

- processar requisições
- aplicar regras de negócio
- gerenciar dados financeiros

## Banco de Dados
Armazena informações como:

- usuários
- transações
- investimentos
- objetivos financeiros

## Autenticação de Usuário
Sistema de login garantindo que cada usuário acesse apenas seus próprios dados.
